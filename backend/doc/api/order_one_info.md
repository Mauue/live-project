# 查询某条预约信息

- 请求方式
    - GET
    
- 请求URL

    `\api\order\<orderid>`
    

- 返回示例
    ```
    {
        "code": 0,
        "data": {
            "id_number": "1231**********3123",
            "name": "1****3",
            "order_num": 3,
            "phone": "123****2312"
        },
        "msg": ""
    }
    ```

- 返回参数说明
    - 
    
- 错误码
|错误码|含义|
|---|---|
|0|成功获取|
|100|预约编号不存在|
|200|抽签未开始|
